# MPR Rollout Dashboard

Internal tool for looking up Employer Partner feature status: Multi-Product Routing (MPR) and Platform Navigation flags, plus product access counts per EP.

**Live:** https://first-cc-project.vercel.app/

## Stack
- React + Vite
- Supabase (Postgres)
- Deployed on Vercel

## Features
- Filter by EP name, MPR status, Platform Nav status, and product
- Sortable table
- Click any row to see per-product user access counts
